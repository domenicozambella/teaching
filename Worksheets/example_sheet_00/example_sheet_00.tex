\begin{exercise}
The language contains only a symbol for a binary relation $e$. The theory $T$ says that $e$ is an equivalence relation and that there is (exactly) $n$ classes containing $n$ elements for each $n\in\omega\sm\{0\}$. Most of the following questions have an almost trivial answer (a few do not).

\begin{itemize}
\item[1.] Prove that every saturated model of $T$ contains a class with infinitely many elements.
\item[2.] Prove every model containing one class with $n$ elements for each $n\in\omega\sm\{0\}$ and infinitely many infinite classes is saturated.   
\item[3.] Prove that the theory $T$ has quantifier elimination.  
\item[4.] Is $T$ is complete?  
\item[5.] Is $T$ countably categorical?
\item[6.] Is $T$ categorical is some uncountable $\lambda$?
\item[7.] What is $\acl(\varnothing)$?
\item[8.] Let $\U\models T$ be saturated. Let $a$ be an element of an finite equivalence class of $e$. What is $\acl\{a\}$?
\item[9.] Let $b$ be an element of an infinite equivalence class. What is $\acl\{b\}$?
\item[10.] If $b$ is like above. What is the orbit of $b$ under $\Aut(\U)$?
\item[11.] Is $T$ strongly minimal?
\end{itemize}
\end{exercise}
